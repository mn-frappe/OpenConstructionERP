Privacy Policy

Last updated: April 6, 2026

1. Who We Are

OpenConstructionERP is developed by DataDrivenConstruction, founded by Artem Boiko. Contact: info@datadrivenconstruction.io

2. Data We Collect

When you use OpenConstructionERP, we may collect:

  • Account data: email address, name, password (hashed)
  • Project data: all data you create within the application (BOQ, documents, estimates) is stored on your server or our demo server
  • Usage data: page views, feature usage (anonymized, for improving the product)
  • Uploaded files: CAD/BIM files, PDFs, photos — stored on the server you deploy to

3. How We Use Your Data

  • To provide the OpenConstructionERP service
  • To improve the product and fix bugs
  • To send important service notifications (security updates, breaking changes)
  • We do NOT sell your data to third parties
  • We do NOT use your project data for AI training

4. Data Storage

Self-hosted: All data stays on your server. We have no access to it.

Demo server: Data on our demo server (openconstructionerp.com/demo) is periodically cleaned and not backed up. Do not store production data there.

5. Third-Party Services

  • AI Providers: If you configure AI (OpenAI, Anthropic, Google), your prompts are sent to their APIs. Review their privacy policies.
  • FormSubmit: Feedback form submissions are processed via FormSubmit.co
  • GitHub: CAD converter downloads come from GitHub releases

6. Your Rights

You have the right to:

  • Access your data (export via API or UI)
  • Delete your account and all associated data
  • Request data portability (all data is exportable as JSON/CSV/Excel)

7. Cookies

We use only essential cookies (session token, language preference). No tracking cookies. No advertising cookies.

8. Security

Passwords are hashed with bcrypt. API access requires JWT authentication. All data transfers use HTTPS. The application source code is open-source (AGPL-3.0) for security auditing.

9. Changes

We may update this policy. Changes will be posted on this page with a new "Last updated" date.

10. Contact

For privacy questions: info@datadrivenconstruction.io
